# SmartTrack 智能试车场综合管理系统

## 项目简介

SmartTrack 是一套面向现代智能试车场的综合管理系统，帮助试车场实现数字化与自动化的预约、车辆调度、试车流程管理、智能监控、数据统计等核心业务。项目采用 Vercel 云部署，前后端一体化、极致开发体验与高可用性。

## 主要功能模块

1. **用户与身份认证**
   - 支持邮箱、社交账号登录（集成 Auth.js/NextAuth.js，兼容第三方认证）
   - 角色权限（试车员/管理员/访客等）
   - 用户信息与安全管理

2. **试车场预约/排班管理**
   - 时间段试车场在线预约
   - 排班/冲突自动检测
   - 活动日历/历史预约查询

3. **车辆全生命周期管理**
   - 车辆登记、档案维护
   - 状态跟踪（在场/离场/维修）
   - 试车车辆分配/调度/调换查询

4. **试车任务与流程配置**
   - 任务发布、分派及进度跟踪
   - 审核、反馈及结案流程
   - 试车任务记录及可视化

5. **智能设备数据接入与监控**
   - 支持视频摄像头、传感器、采集终端数据实时接入
   - 试车实时监控看板
   - 设备状态及报警处理

6. **数据统计与报告**
   - 车辆与任务的历史统计分析
   - 场地利用率、试车绩效图表
   - 数据导出及自定义报表

7. **消息通知与自动提醒**
   - 预约、任务、异常多渠道推送（站内、Email等）
   - 任务到期、设备告警智能提醒

8. **系统设置与运维**
   - 试车场基础参数及时间段配置
   - 日志、审计与平台维护
   - 权限与安全强化

## 技术栈说明

### 前端

- **Next.js**（React 框架，Vercel 原生支持）
- **Tailwind CSS**（原子化样式，快速响应式开发）
- **SandcnUI**（高效美观的 React UI 组件库）
- **TanStack Table**（强大表格功能，数据统计/管理必备）
- **Auth.js/NextAuth.js**（认证/登录，兼容第三方登录与 JWT）
- **React Query / TanStack Query**（异步数据管理与状态同步）
- **Zustand（推荐）/Redux**（简洁高效状态管理）
- **React Hook Form**（复杂表单处理）
- **Axios**（API 请求利器）
- **Day.js**（日期时间处理）
- **ECharts / Recharts**（可视化统计图表）
- 支持 SSR/SSG，适配 Vercel 极致性能

### 后端及数据

- **Next.js API Route（优先）/Express.js**（API 层集成，前后端一体/可拔插扩展）
- **MongoDB**（非关系型数据库，支持多样数据结构）
- **Mongoose**（MongoDB 对象建模及数据验证）
- **JWT、Auth.js/NextAuth.js、Passport.js**（多种认证方式安全接入）

### 部署架构

- **Vercel**（前端/后端一体部署，CI/CD 无缝）
- **MongoDB Atlas**（托管云数据库）
- 持续集成，自动化部署

## 快速启动

1. **克隆项目**
   ```bash
   git clone https://github.com/litantai/SmartTrack.git
   cd SmartTrack
   ```

2. **配置环境变量**
   - 新建 `.env.local` 文件
   - 配置数据库连接、认证密钥等（参考 `.env.example`）

3. **本地开发模式启动**
   ```bash
   npm install
   npm run dev
   ```

4. **一键部署到 Vercel**
   - 直接在 Vercel Dashboard 选择仓库自动部署
   - 或同步 GitHub 推送，Vercel 自动 CI/CD

5. **在线体验与维护**
   - 支持在线升级，环境无缝发布
   - 数据安全与自动容灾保障
